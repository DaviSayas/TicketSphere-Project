OPÇÃO 2: SCRIPT AUTOMÁTICO (Um Clique)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Crie um arquivo: iniciar-helpdesk.ps1
2. Cole o conteúdo do arquivo INSTRUCOES_POWERSHELL.md (secção "Método 3")
3. Na PowerShell, execute:
   
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   .\iniciar-helpdesk.ps1

✓ Dois terminais abrem automaticamente

🛠️ TROUBLESHOOTING:

❌ "Porta já está em uso"
✓ Mudar porta: python -m uvicorn app.main:app --reload --port 8001

❌ "venv não está ativado"
✓ Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
✓ .\venv\Scripts\activate

❌ "Python não encontrado"
✓ python --version
✓ Reinstalar Python ou adicionar ao PATH

💡 DICAS:

• VS Code: Use Ctrl + Shift + ` para abrir novo terminal

• Recarregar app: Ctrl + F5 (hard refresh no navegador)
